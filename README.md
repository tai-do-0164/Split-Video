## LightSplit – Ultra-Lightweight Node.js Video Splitter
Splitting videos has never been faster or easier! LightSplit is a super-light Node.js package that starts instantly and lets you split videos quickly and efficiently. Perfect for web projects, backend processing, or any workflow that needs fast, reliable video handling.

- ✅ Instant startup
- ✅ Ultra-light and easy to integrate
- ✅ Flexible video splitting 

Get started with LightSplit and experience lightning-fast video processing!
